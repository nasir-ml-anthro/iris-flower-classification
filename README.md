# 🌸 Iris Flower Classification using KNN
### Hybrid Approach: Machine Learning & Behavioral Patterns

This project is my first step into the world of **Data Science** and **Machine Learning**. As a Zoology student and an aspiring **Digital Anthropologist**, I am exploring how we can use algorithms like **K-Nearest Neighbors (KNN)** to classify biological data and bridge the gap between human observation and machine precision.

## 🚀 Project Overview
This repository contains a simple yet powerful implementation of the **KNN algorithm** to classify different species of the Iris flower based on their physical attributes.

## 🛠️ Tech Stack
* **Language:** Python 3.13.12
* **Libraries:** Pandas, Scikit-learn
* **Algorithm:** K-Nearest Neighbors (KNN)

## 📂 Files in this Repository
* `iris_knn.py`: The main Python script containing the model.
* `My first project of AI.txt`: Documentation of the initial thought process.
